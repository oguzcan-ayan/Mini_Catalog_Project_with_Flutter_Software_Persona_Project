# AppTrendMarket App

Bu proje bir e-ticaret uygulamalarından ilham alınıp oluşturulan; kullanıcının uygulama içerisindeki ürünleri görmesini, detayına bakabilmesini, sepete eklemesini ve en sonunda aldığı ürünlerin tamamını görmesini sağlayan bir mobil uygulamadır.

This project is a mobile application inspired by e-commerce applications; it allows users to view products within the application, examine their details, add them to their cart, and finally see all the products they have purchased.
